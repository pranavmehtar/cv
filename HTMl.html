This Blog will tell you about PHP Data Types.

Data Type is thing which can define what type of Data present in the page

  1. Scalar Data

    • Integer Data
    • Float Data
    • String Data
    • Boolean Data

  2. Compound Data

    • Array
    • Object

  3. Special Data

    • Resource
    • Null

Scalar Data :-
1) Integer :- This is int data type which store the value of integer whic does not have decimal point, for eg. 2,-4,6.
2) Float :- Float type variable stores the value of number which have floating point.
3)str :- This type of variable store the text data. The data which is sequence of data, either it is in double quotes or single qoutes.
